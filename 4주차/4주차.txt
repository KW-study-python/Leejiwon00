1. 지도학습
- 인공지능을 누군가가 직접 가르치고 이끄는 학습 방법
- 데이터 중에서도 정답이 있는 데이터를 이용하여 학습
- 사례: 분류( 이진분류 다중분류), 회귀

2. 비지도 학습
- 정답이 없는 데이터를 사용하여 그 특징을 찾아 스스로 구분
- 새로운 데이터가 들어왔을 때, 그 데이터가 어떤 그룹에 속하는지 스스로 판단
- 사례: 군집화, 차원 축소

3. 티처블 머신
- 구글에서 개발한 인공지능 개발 체험 도구로, 쉽고 간단하게 인공지능을 만들 수 있음
- 머신러닝의 학습 방법 중 지도학습 방법을 사용

4.퀵드로우
- 구글에서 인공지능을 체험할 수 있게 만든 사이트로 낙서 인식 기능

5. 오토드로우
- 퀵드로우와 비슷하지만 사용자가 그린 그림을 인공지능을 통해 업그레이드해주는 기능을 제공
- 퀵드로우 데이터셋 사용

6. 딥러닝
- 사람의 뇌에서 이루어지는 원리를 이용하여 인공지능을 만드는 방식
- 심층 신경망이 학습하는 과정

7.인공 신경망
- 신경망을 사람들이 인공적으로 만든 것
- 신경망의 최소 구성 단위인 뉴런이 다른 뉴런과 연결된 모습을 각각의 층을 사용해 연결
